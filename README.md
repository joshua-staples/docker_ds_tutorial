## Pre-requisits:

- MacOS, Linux, Windows
- Docker Desktop

Check install:

```
docker ps 
```
```
docker version
```
```
docker run hello-world
```
To show all running containers:
```
docker ps -a
```
One last example:
```
docker run docker/whalesay cowsay Hello there!
```
If the docker image doesn't exist locally, then docker will automatically pull it from ![dockerhub.com]
